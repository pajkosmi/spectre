Detailed Description

Numeric initial data loaded from volume data files.

This class can be factory-created in the input file to start an evolution from numeric initial data. It selects the set of GH variables to load from the volume data file (ADM or GH variables), the set of hydro variables, and allows to set constant values for some of the hydro variables. This class mostly combines the gh::NumericInitialData and grmhd::ValenciaDivClean::NumericInitialData classes.
