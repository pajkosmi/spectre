Detailed Description

Intermediate base class implementing a generic Runge-Kutta scheme.

Implements most of the virtual methods of TimeStepper for a generic Runge-Kutta method. From the TimeStepper interface, derived classes need only implement order, error_estimate_order, and stable_step, and should not include the TIME_STEPPER_* macros. All other methods are implemented in terms of a Butcher tableau returned by the butcher_tableau function.
