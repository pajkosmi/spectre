Detailed Description

Checks if the regular field has been received from the worldtube and computes the retarded field for boundary conditions.

Details

This action checks whether values for the regular field have been send by the worldtube. If so, the spatial derivative is converted from the grid to inertial frame and the puncture field is added to it to obtain the retarded field. This is stored in Tags::WorldtubeSolution which is used to formulate boundary conditions in CurvedScalarWave::BoundaryConditions::Worldtube.
