Detailed Description

template<typename DataType, size_t Dim, ::domain::ObjectLabel Label, typename GridCoordsTag, typename OutputCoordsTag, typename Fr = Frame::Inertial>
struct hydro::Tags::MassWeightedCoordsCompute< DataType, Dim, Label, GridCoordsTag, OutputCoordsTag, Fr >

Compute item for TildeD * (coordinates in frame Fr). Label allows us to restrict the data to the x>0 (A) or x<0 (B) plane in grid coordinates. Use ObjectLabel::None for unrestricted data. Can be retrieved using ‘hydro::Tags::MassWeightedCoords’.
