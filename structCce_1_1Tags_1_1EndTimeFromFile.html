Detailed Description

Represents the final time of a bounded CCE evolution, determined either from option specification or from the file.

Details

If no end time is specified in the input file (so the option OptionTags::EndTime is set to "Auto"), this will find the end time from the provided H5 file. If OptionTags::EndTime takes any other value, it will be used directly as the final time for the CCE evolution instead.
