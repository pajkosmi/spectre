Detailed Description

template<size_t Dim>
struct ScalarAdvection::subcell::DgInitialDataTci< Dim >

The troubled-cell indicator run on DG initial data to see if we need to switch to subcell.

Apply 1) the two-mesh relaxed discrete maximum principle to the scalar field \(U\), and 2) the Persson TCI to the scalar field \(U\) if the \(\max(|U|)\) on the DG grid is greater than tci_options.u_cutoff.
