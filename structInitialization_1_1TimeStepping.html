Detailed Description

template<typename Metavariables, bool UsingLts>
struct Initialization::TimeStepping< Metavariables, UsingLts >

Initialize items related to time stepping.

Details

See the type aliases defined below for what items are added to the GlobalCache, MutableGlobalCache, and DataBox and how they are initialized

Since the evolution has not started yet, initialize the state before the initial time. So Tags::TimeStepId is undefined at this point, and Tags::Next<Tags::TimeStepId> is the initial time.
