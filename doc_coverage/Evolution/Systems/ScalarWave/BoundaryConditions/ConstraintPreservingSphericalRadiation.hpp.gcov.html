      56             :  *
      57             :  * The constraint-preserving part of the boundary condition imposes the
      58             :  * following condition on the time derivatives of the characteristic fields:
      59             :  *
      60             :  * \f{align*}{
      61             :  *   d_tw^\Psi&\to d_tw^{\Psi}+\lambda_{\Psi}n^i\mathcal{C}_i, \\
      62             :  *   d_tw^0_i&\to d_tw^{0}_i+\lambda_{0}n^jP^k_i\mathcal{C}_{ik},
      63             :  * \f}
      64             :  *
      65             :  * where
      66             :  *
      67             :  * \f{align*}{
      68             :  * P^k{}_i=\delta^k_i-n^kn_i
      69             :  * \f}
      70             :  *
      71             :  * projects a tensor onto the spatial surface to which \f$n_i\f$ is normal, and
      72             :  * \f$d_t w\f$ is the evolved to characteristic field transformation applied to
      73             :  * the time derivatives of the evolved fields. That is,
      74             :  *
      75             :  * \f{align*}{
      76             :  * d_t w^\Psi&=\partial_t \Psi, \\
      77             :  * d_t w_i^0&=(\delta^k_i-n^k n_i)\partial_t \Phi_k, \\
      78             :  * d_t w^{\pm}&=\partial_t\Pi\pm n^k\partial_t\Phi_k - \gamma_2\partial_t\Psi.
      79             :  * \f}
      80             :  *
      81             :  * The constraints are defined as:
      82             :  *
      83             :  * \f{align*}{
      84             :  *  \mathcal{C}_i&=\partial_i\Psi - \Phi_i=0, \\
      85             :  *  \mathcal{C}_{ij}&=\partial_{[i}\Phi_{j]}=0
      86             :  * \f}
      87             :  *
      88             :  * Radiation boundary conditions impose a condition on \f$\Pi\f$ or its time
      89             :  * derivative. We denote the boundary condition value of the time derivative of
      90             :  * \f$\Pi\f$ by \f$\partial_t\Pi^{\mathrm{BC}}\f$. With this, we can impose
      91             :  * boundary conditions on the time derivatives of the evolved variables as
      92             :  * follows:
      93             :  *
      94             :  * \f{align*}{
      95             :  * \partial_{t} \Psi&\to\partial_{t}\Psi +
      96             :  *                    \lambda_\Psi n^i \mathcal{C}_i, \\
      97             :  * \partial_{t}\Pi&\to\partial_{t}\Pi-\left(\partial_t\Pi
      98             :  *                  - \partial_t\Pi^{\mathrm{BC}}\right)
      99             :  *                  +\gamma_2\lambda_\Psi n^i \mathcal{C}_i
     100             :  *                  =\partial_t\Pi^{\mathrm{BC}}
     101             :  *                  +\gamma_2\lambda_\Psi n^i \mathcal{C}_i, \\
     102             :  * \partial_{t}\Phi_i&\to\partial_{t}\Phi_i+ 2 \lambda_0n^j \mathcal{C}_{ji}.
     103             :  * \f}
     104             :  *
     105             :  * Below we assume the normal vector \f$n^i\f$ is the radial unit normal vector.
     106             :  * That is, we assume the outer boundary is spherical. A Sommerfeld
     107             :  * \cite Sommerfeld1949 radiation condition is given by
     108             :  *
     109             :  * \f{align*}{
     110             :  *  \partial_t\Psi=n^i\Phi_i
     111             :  * \f}

     113             :  * Or, assuming that \f$\partial_tn^i=0\f$ (or is very small),
     114             :  *
     115             :  * \f{align*}{
     116             :  *  \partial_t\Pi^{\mathrm{BC}}=n^i\partial_t\Phi_i
     117             :  * \f}
     118             :  *
     119             :  * The Bayliss-Turkel \cite BaylissTurkel boundary conditions are given by:
     120             :  *
     121             :  * \f{align*}{
     122             :  *  \prod_{l=1}^m\left(\partial_t + \partial_r + \frac{2l-1}{r}\right)\Psi=0
     123             :  * \f}
     124             :  *
     125             :  * The first-order form is

     129             :  * \f}

     132             :  *
     133             :  * The second-order boundary condition is given by,
     134             :  *

     137             :  *   &=\left(\partial_t\partial_r + \partial_r\partial_t +
     138             :  *     \partial_r^2+\frac{4}{r}\partial_t

     149             :  *
