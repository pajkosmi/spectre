     276             :  * If the spherically symmetric metric is written as
     277             :  *
     278             :  * \f[
     279             :  * ds^2 = - e^{2 \Phi_t} dt^2 + e^{2 \Phi_r} dr^2 + e^{2 \Phi_\Omega} r^2

     288             :  * \alpha &= e^{\Phi_t} \\

     290             :  * \gamma_{ij} &= \delta_{ij} e^{2 \Phi_r} + \delta_{im} \delta_{jn}
