      72             :  * called `RotNS`.
      73             :  *
      74             :  * The metric in spherical coordinates is given by \cite Cook1992
      75             :  *
      76             :  * \f{align}{
      77             :  *   ds^2=-e^{\gamma+\rho}dt^2+e^{2\alpha}(dr^2+r^2d\theta^2)
      78             :  *   +e^{\gamma-\rho}r^2\sin^2(\theta)(d\phi-\omega dt)^2.
      79             :  * \f}
      80             :  *
      81             :  * We use rotation about the \f$z\f$-axis. That is,
      82             :  *
      83             :  * \f{align}{
      84             :  *   g_{tt}
      85             :  *   &=-e^{\gamma+\rho} + e^{\gamma-\rho}r^2\sin^2(\theta)\omega^2 \\
      86             :  *   g_{rr}
      87             :  *   &=e^{2\alpha} \\
      88             :  *   g_{\theta\theta}
      89             :  *   &=e^{2\alpha}r^2 \\
      90             :  *   g_{\phi\phi}
      91             :  *   &=e^{\gamma-\rho}r^2\sin^2(\theta) \\
      92             :  *   g_{t\phi}
      93             :  *   &=-e^{\gamma-\rho}r^2\sin^2(\theta)\omega.
      94             :  * \f}
      95             :  *
      96             :  * We can transform from spherical to Cartesian coordinates using
      97             :  *
      98             :  * \f{align}{
      99             :  *   \label{eq:Jacobian}
     100             :  *   \frac{\partial (r,\theta,\phi)}{\partial (x,y,z)}=
     101             :  *   \begin{pmatrix}
     102             :  *     \cos(\phi) \sin(\theta) & \sin(\theta)\sin(\phi) & \cos(\theta) \\
     103             :  *     \tfrac{\cos(\phi)\cos(\theta)}{r} & \tfrac{\sin(\phi)\cos(\theta)}{r}
     104             :  *     & -\tfrac{\sin(\theta)}{r} \\
     105             :  *     -\tfrac{\sin(\phi)}{r\sin(\theta)} & \tfrac{\cos(\phi)}{r\sin(\theta)}
     106             :  *     & 0
     107             :  *   \end{pmatrix}
     108             :  * \f}
     109             :  *
     110             :  * and
     111             :  *
     112             :  * \f{align}{
     113             :  *   \frac{\partial (x,y,z)}{\partial (r,\theta,\phi)}=
     114             :  *   \begin{pmatrix}
     115             :  *     \cos(\phi) \sin(\theta) & r\cos(\phi)\cos(\theta) &
     116             :  * -r\sin(\theta)\sin(\phi)
     117             :  *     \\
     118             :  *     \sin(\theta)\sin(\phi) & r\sin(\phi)\cos(\theta) &
     119             :  * r\cos(\phi)\sin(\theta)
     120             :  *     \\
     121             :  *     \cos(\theta) & -r\sin(\theta) & 0
     122             :  *   \end{pmatrix}
     123             :  * \f}

     142             :  *   \beta^y &= \partial_\phi y \beta^\phi = -\cos(\phi)\omega r \sin(\theta),
