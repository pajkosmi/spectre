      41             :  * \brief Kerr black hole in Spherical Kerr-Schild coordinates
      42             :  *
      43             :  * ## Introduction
      44             :  *
      45             :  * Given a Kerr-Schild (KS) black hole system, we denote the coordinate system
      46             :  * using \f$\{t,x,y,z\}\f$. In the transformed system, Spherical Kerr-Schild
      47             :  * (Spherical KS), we denote the coordinate system using
      48             :  * \f$\{t,\bar{x},\bar{y},\bar{z}\}\f$. Further, when considering indexed
      49             :  * objects, we will use Greek and Latin indices with the standard convention
      50             :  * \f$(\mu=0,1,2,3\f$ and \f$i=1,2,3\f$ respectively), but we will use a bar to
      51             :  * denote that a given index is in reference to the Spherical KS coordinate
      52             :  * system (i.e. \f$i\f$ vs. \f$\bar{\imath}\f$ and \f$\mu\f$ vs.
      53             :  * \f$\bar{\mu}\f$).
      54             :  *
      55             :  * ## Spin in the z direction
      56             :  *
      57             :  * ### The Transformation
      58             :  *
      59             :  * The Boyer-Lindquist radius for KS with its spin in the \f$z\f$ direction is
      60             :  * defined by
      61             :  *
      62             :  * \f{align}{
      63             :  *    \frac{x^2 + y^2}{r^2 + a^2} + \frac{z^2}{r^2} = 1,
      64             :  * \f}
      65             :  *
      66             :  * or equivalently,
      67             :  *
      68             :  * \f{align}{
      69             :  *    r^2 &= \frac{1}{2}\left(x^2+y^2+z^2-a^2\right) +
      70             :  *    \left(\frac{1}{4}\left(x^2+y^2+z^2-a^2\right)^2 + a^2z^2\right)^{1/2}.
      71             :  * \f}
      72             :  *
      73             :  * The Spherical KS coordinates
      74             :  *
      75             :  * \f{align}{
      76             :  *    \vec{\bar{x}} = x^{\bar{\imath}} = x_{\bar{\imath}} =
      77             :  *    (\bar{x},\bar{y},\bar{z}),
      78             :  * \f}
      79             :  *
      80             :  * and KS coordinates
      81             :  *
      82             :  * \f{align}{
      83             :  *    \vec{x} = x^{i} = x_{i} = (x,y,z),
      84             :  * \f}
      85             :  *
      86             :  * are related by
      87             :  *
      88             :  * \f{align}{
      89             :  *    \left(\frac{\bar{x}}{r},\frac{\bar{y}}{r},\frac{\bar{z}}{r}\right) \equiv
      90             :  *    \left(\frac{x}{\rho},\frac{y}{\rho},\frac{z}{r}\right),
      91             :  * \f}
      92             :  *
      93             :  * where we have defined
      94             :  *
      95             :  * \f{align}{
      96             :  *   \rho \equiv r^2 + a^2. \label{eq:rho}
      97             :  * \f}
      98             :  *
      99             :  * Therefore, we have that \f$r\f$ satisfies the equation for a sphere in
     100             :  * Spherical KS
     101             :  *
     102             :  * \f{align}{
     103             :  *    r^2 = \vec{\bar{x}}\cdot\vec{\bar{x}} = \bar{x}^2 + \bar{y}^2 +
     104             :  *    \bar{z}^2.
     105             :  * \f}
     106             :  *
     107             :  * It is clear to see that the Spherical KS radius coincides with the
     108             :  * Boyer-Lindquist radius.
     109             :  *
     110             :  * ## Spin in an Arbitrary Direction
     111             :  *
     112             :  * Given that the remaining important quantities take forms that are easily
     113             :  * specialized to the \f$z\f$ spin case, we instead focus on the general case
     114             :  * for brevity.
     115             :  *
     116             :  * ### The Transformation
     117             :  *
     118             :  * The Boyer-Lindquist radius for KS with spin in an arbitrary direction is
     119             :  * defined by
     120             :  *
     121             :  * \f{align}{
     122             :  *    r^2 = \frac{1}{2}\left(\vec{x}\cdot\vec{x}-a^2\right) +
     123             :  *    \left(\frac{1}{4}\left(\vec{x}\cdot\vec{x}-a^2\right)^2 +
     124             :  *    \left(\vec{a}\cdot\vec{x}\right)^2\right)^{1/2}.
     125             :  * \f}
     126             :  *
     127             :  * Then, defining two transformation matrices \f$Q^{\bar{\imath}}{}_{j}\f$ and
     128             :  * \f$P^{j}{}_{\bar{\imath}}\f$ as
     129             :  *
     130             :  * \f{align}{
     131             :  *    Q^{\bar{\imath}}{}_{j} &= \frac{r}{\rho}\delta^{\bar{\imath}}{}_{j} +
     132             :  *    \frac{1}{(\rho + r)\rho}a^{\bar{\imath}}a_{j}, \\
     133             :  *    P^{j}{}_{\bar{\imath}} &= \frac{\rho}{r}\delta^{j}{}_{\bar{\imath}} -
     134             :  *    \frac{1}{(\rho + r)r}a^{j}a_{\bar{\imath}},
     135             :  * \f}
     136             :  *
     137             :  * where the definition of \f$\rho\f$ is identical to Eq. \f$(\ref{eq:rho})\f$,
     138             :  * such that
     139             :  *
     140             :  * \f{align}{
     141             :  *   Q^{\bar{\imath}}{}_{j}x^{j} &= x^{\bar{\imath}}, \\
     142             :  *   P^{j}{}_{\bar{\imath}}x^{\bar{\imath}} &= x^{j}.
     143             :  * \f}
     144             :  *
     145             :  * We again recover that \f$r\f$ satisfies the equation for a sphere in
     146             :  * Spherical KS
     147             :  *
     148             :  * \f{align}{
     149             :  *   r^2 = \vec{\bar{x}}\cdot\vec{\bar{x}} = \bar{x}^2+\bar{y}^2+\bar{z}^2,
     150             :  * \f}
     151             :  *
     152             :  * and we also have that
     153             :  *
     154             :  * \f{align}{
     155             :  *   \vec{a}\cdot\vec{\bar{x}} = \vec{a}\cdot\vec{x}.
     156             :  * \f}
     157             :  *
     158             :  * Note that \f$Q^{\bar{\imath}}{}_{ j}\f$ and \f$P^{j}{}_{\bar{\imath}}\f$
     159             :  * satisfy
     160             :  *
     161             :  * \f{align}{
     162             :  *   P^{i}{}_{\bar{\jmath}}\,Q^{\bar{\jmath}}{}_{ k} = \delta^{i}{}_{ k}.
     163             :  * \f}
     164             :  *
     165             :  * The Jacobian is then given by
     166             :  *
     167             :  * \f{align}{
     168             :  *   T^{i}{}_{\bar{\jmath}} = \partial_{\bar{\jmath}}\,x^{i} =
     169             :  *   \frac{\partial x^i}{\partial x^{\bar{\jmath}}},
     170             :  * \f}
     171             :  *
     172             :  * while its inverse is given by
     173             :  *
     174             :  * \f{align}{
     175             :  *   S^{\bar{\jmath}}{}_{ i} = \partial_{i}\,x^{\bar{\jmath}} =
     176             :  *   \frac{\partial x^{\bar{\jmath}}}{\partial x^i},
     177             :  * \f}
     178             :  *
     179             :  * which in turn satisfies
     180             :  *
     181             :  * \f{align}{
     182             :  *   T^{i}{}_{\bar{\jmath}}\,S^{\bar{\jmath}}{}_{ k} &= \delta^i_{\; k}.
     183             :  * \f}
     184             :  *
     185             :  * ### The Metric
     186             :  *
     187             :  * A KS coordinate system is defined by
     188             :  *
     189             :  * \f{align}{
     190             :  *   g_{\mu\nu} = \eta_{\mu\nu} + 2Hl_{\mu}l_{\nu},
     191             :  * \f}
     192             :  *
     193             :  * where \f$H\f$ is a scalar function of the coordinates, \f$\eta_{\mu\nu}\f$ is
     194             :  * the Minkowski metric, and \f$l^\mu\f$ is a null vector. Note that the inverse
     195             :  * of the spacetime metric is given by
     196             :  *
     197             :  * \f{align}{
     198             :  *   g^{\mu\nu} = \eta^{\mu\nu} - 2Hl^{\mu}l^{\nu}.
     199             :  * \f}
     200             :  *
     201             :  * The scalar function \f$H\f$ takes the form
     202             :  *
     203             :  * \f{align}{
     204             :  *   H = \frac{Mr^3}{r^4 + \left(\vec{a}\cdot\vec{x}\right)^{2}},
     205             :  * \f}
     206             :  *
     207             :  * where \f$M\f$ is the mass, while the spatial part of the null vector takes
     208             :  * the form
     209             :  *
     210             :  * \f{align}{
     211             :  *   l_{i} = l^{i} = \frac{r\vec{x} - \vec{a}\times\vec{x} +
     212             :  *   \frac{(\vec{a}\cdot\vec{x})\vec{a}}{r}}{\rho^2}.
     213             :  * \f}
     214             :  *
     215             :  * Note that the full spacetime form of the null vector is
     216             :  *
     217             :  * \f{align}{
     218             :  *   l_{\mu} &= (-1,l_{i}), & l^{\mu} &= (1,l^{i}).
     219             :  * \f}
     220             :  *
     221             :  * Transforming the KS spatial metric then yields the following Spherical KS
     222             :  * spatial metric
     223             :  *
     224             :  * \f{align}{
     225             :  *   \gamma_{\bar{\imath}\bar{\jmath}} &=
     226             :  *   \gamma_{mn}T^{m}{}_{\bar{\imath}}\,T^{n}{}_{\bar{\jmath}}, \nonumber \\
     227             :  *   &= \eta_{mn}T^{m}{}_{\bar{\imath}}\,T^{n}{}_{\bar{\jmath}} +
     228             :  *   2Hl_{m}l_{n}T^{m}{}_{\bar{\imath}}\,T^{n}{}_{\bar{\jmath}}, \nonumber \\
     229             :  *   &= \eta_{\bar{\imath}\bar{\jmath}} + 2Hl_{\bar{\imath}}l_{\bar{\jmath}}.
     230             :  * \f}
     231             :  *
     232             :  * The transformed spacetime Minkowski metric is given by
     233             :  *
     234             :  * \f{align}{
     235             :  *   \eta_{\bar{\mu}\bar{\nu}} = (-1)\otimes\eta_{\bar{\imath}\bar{\jmath}},
     236             :  * \f}
     237             :  *
     238             :  * and the transformed spacetime null vector is given by
     239             :  *
     240             :  * \f{align}{
     241             :  *   l_{\bar{\mu}} &= (-1,l_{\bar{\imath}}), &
     242             :  *   l^{\bar{\mu}} &= (1,l^{\bar{\imath}}).
     243             :  * \f}
     244             :  *
     245             :  * Therefore, the Spherical KS spacetime metric is
     246             :  *
     247             :  * \f{align}{
     248             :  *   g_{\bar{\mu}\bar{\nu}} = \eta_{\bar{\mu}\bar{\nu}} +
     249             :  *   2Hl_{\bar{\mu}}l_{\bar{\nu}}.
     250             :  * \f}
     251             :  *
     252             :  * Further, we have that the lapse in Spherical KS is given by
     253             :  *
     254             :  * \f{align}{
     255             :  *   \alpha = \left(1 + 2H\right)^{-1/2},
     256             :  * \f}
     257             :  *
     258             :  * and the shift in Spherical KS by
     259             :  *
     260             :  * \f{align}{
     261             :  *   \beta^{\bar{\imath}} &= -\frac{2Hl^{t}l^{\bar{\imath}}}{1 + 2Hl^{t}l^{t}} =
     262             :  *   -2H\alpha^{2}l^{t}l^{\bar{\imath}}, & \beta_{\bar{\imath}} &=
     263             :  *  -2Hl_{t}l_{\bar{\imath}}.
     264             :  * \f}
     265             :  *
     266             :  * ### Derivatives
     267             :  *
     268             :  * The derivatives of the preceding quantities are
     269             :  *
     270             :  * \f{align}{
     271             :  *   \frac{\partial r}{\partial x^{i}} &= \frac{r^{2}x_{i} +
     272             :  *   \left(\vec{a}\cdot\vec{x}\right)a_{i}}{rs}, \\

     301             :  *   4Hl^{\bar{\imath}}\alpha^{4}\partial_{\bar{k}}H,
     302             :  * \f}
     303             :  *
     304             :  * where we have defined \f$s\f$ as
     305             :  *
     306             :  * \f{align}{

     308             :  *   \label{eq: s_number}
     309             :  * \f}
     310             :  *
     311             :  * ## Code
     312             :  *
     313             :  * While the previous sections described the relevant physical quantities, the
     314             :  * actual files make use of internally defined objects to ease the computation
     315             :  * of the Jacobian, the inverse Jacobian, and their corresponding derivatives.
     316             :  * Therefore, we now list these intermediary objects as well as how they

     318             :  * definition found in the code (all for arbitrary spin).
     319             :  *
     320             :  * ### Helper Matrices
     321             :  *
     322             :  * The intermediary objects used to define the various Jacobian objects, the
     323             :  * so-called "helper matrices", are defined below.
     324             :  *
     325             :  * \f{align}{

     328             :  *   a^{i}a_{\bar{k}}\right), \\

     353             :  *
     354             :  * where \f$s\f$ is defined identically to Eq. \f$(\ref{eq: s_number})\f$.

     367             :  *   \left(G_2\right)^{\bar{n}}{}_{ j}, \\
