      94             :  *   can optionally be used in this layer. This allows the domain to extend to
      95             :  *   large radial distances from the compact objects. This layer can be
      96             :  *   h-refined radially, creating a layer of multiple concentric spherical

      98             :  *

     102             :  *   and Object B is located to the left of the origin.

     106             :  *   the center of mass is located at x=0.

     108             :  *    object with a single cartesian cube. This is less efficient, but allows
