     128             :  * spherical shells, each composed of six wedges, can be added with the
     129             :  * 'RadialPartitioning' option.

     132             :  * sphere."
     133             :  *
     134             :  * #### Inner cube sphericity
     135             :  * The inner cube is a BulgedCube except if the inner cube sphericity is
     136             :  * exactly 0. Then an Equiangular or Affine map is used (depending on if it's
     137             :  * equiangular or not) to avoid a root find in the BulgedCube map.
     138             :  *
     139             :  * #### Time dependent maps
     140             :  * There are two ways to add time dependent maps to the Sphere domain
     141             :  * creator. In the input file, these are specified under the
     142             :  * `TimeDependentMaps:` block.
     143             :  *
     144             :  * ##### TimeDependence
     145             :  * You can use a simple TimeDependence (e.g.
     146             :  * `domain::creators::time_dependence::UniformTranslation` or
     147             :  * `domain::creators::time_dependence::RotationAboutZAxis`) to add time
     148             :  * dependent maps. This method will add the same maps to all blocks in the
     149             :  * domain. This method can be used with an inner cube or with an excision
     150             :  * surface.
     151             :  *
     152             :  * ##### Hard-coded time dependent maps
     153             :  * The Sphere domain creator also has the option to use some hard coded time
     154             :  * dependent maps that may be useful in certain scenarios. This method adds the
     155             :  * maps in `domain::creators::sphere::TimeDependentMapOptions` to the domain.
     156             :  * Currently, the first (inner-most) shell has maps between `Frame::Grid`,
     157             :  * `Frame::Distorted`, and `Frame::Inertial` while all subsequent shells only
     158             :  * have maps between `Frame::Grid` and `Frame::Inertial`.
     159             :  *
     160             :  * \note You can only use hard-coded time dependent maps if you have an excision
     161             :  * surface. You cannot have a inner cube.
     162             :  *
     163             :  * ##### None
     164             :  * To not have any time dependent maps, pass a `std::nullopt` to appropriate
