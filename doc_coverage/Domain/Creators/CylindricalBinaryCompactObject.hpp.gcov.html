      82             :  * Note that Figure 20 of \cite Buchman:2012dw illustrates additional

      84             :  * of Figure 20 indicates that there are additional spherical shells
      85             :  * outside the "CA" and "CB" blocks; `CylindricalBinaryCompactObject`
      86             :  * has these extra shells inside "EA" only if the option `IncludeInnerSphereA`
      87             :  * is true, it has the extra shells inside "EB" only if the option
      88             :  * `IncludeInnerSphereB` is true, and it has the extra shells outside
      89             :  * "CA" and "CB" only if `IncludeOuterSphere` is true.
      90             :  * If the shells are absent, then the "EA" and "EB"
      91             :  * blocks extend to the excision boundaries and the "CA" and "CB" blocks
      92             :  * extend to the outer boundary.
      93             :  *
      94             :  * The Blocks are named as follows:
      95             :  * - Each of CAFilledCylinder, EAFilledCylinder, EBFilledCylinder,
      96             :  *   MAFilledCylinder, MBFilledCylinder, and CBFilledCylinder consists
      97             :  *   of 5 blocks, named 'Center', 'East', 'North', 'West', and
      98             :  *   'South', so an example of a valid block name is
      99             :  *   'CAFilledCylinderCenter'.
     100             :  * - Each of CACylinder, EACylinder, EBCylinder, and CBCylinder
     101             :  *   consists of 4 blocks, named 'East', 'North', 'West', and 'South',
     102             :  *   so an example of a valid block name is 'CACylinderEast'.
     103             :  * - The Block group called "Outer" consists of all the CA and CB blocks. They
     104             :  *   all border the outer boundary if `IncludeOuterSphere` is false.
     105             :  * - If `IncludeOuterSphere` is true, then there are more blocks named
     106             :  *   OuterSphereCAFilledCylinder, OuterSphereCBFilledCylinder,
     107             :  *   OuterSphereCACylinder, and OuterSphereCBCylinder.
     108             :  *   These are in a Block group called "OuterSphere",
     109             :  *   and all of these border the outer boundary.
     110             :  * - The Block group called "InnerA" consists of all the EA, and MA
     111             :  *   blocks. They all border the inner boundary "A" if
     112             :  *   `IncludeInnerSphereA` is false.
     113             :  * - If `IncludeInnerSphereA` is true, then there are new blocks
     114             :  *   InnerSphereEAFilledCylinder, InnerSphereMAFilledCylinder, and
     115             :  *   InnerSphereEACylinder. These are in a Block group called "InnerSphereA",
     116             :  *   and all of these border the inner excision boundary "A".
     117             :  * - The Block group called "InnerB" consists of all the EB, and MB
     118             :  *   blocks. They all border the inner boundary "B" if
     119             :  *   `IncludeInnerSphereB` is false.
     120             :  * - If `IncludeInnerSphereB` is true, then there are new blocks
     121             :  *   InnerSphereEBFilledCylinder, InnerSphereMBFilledCylinder, and
     122             :  *   InnerSphereEBCylinder. These are in a Block group called "InnerSphereB",
     123             :  *   and all of these border the inner excision boundary "B".
     124             :  *
     125             :  * If \f$c_A\f$ and \f$c_B\f$ are the input parameters center_A and
     126             :  * center_B, \f$r_A\f$ and \f$r_B\f$ are the input parameters radius_A and
     127             :  * radius_B, and \f$R\f$ is the outer boundary radius, we demand the
     128             :  * following restrictions on parameters:
     129             :  * - \f$c_A^0>0\f$; this is a convention to simplify the code.
     130             :  * - \f$c_B^0<0\f$; this is a convention to simplify the code.
     131             :  * - \f$|c_A^0|\le|c_B^0|\f$. We should roughly have \f$r_A c_A^0 + r_B c_B^0\f$
     132             :  *   close to zero; that is, for BBHs (where \f$r_A\f$ is roughly twice the
     133             :  *   mass of the heavier object A, and \f$r_B\f$ is roughly twice the mass
     134             :  *   of the lighter object B) the center of mass should be roughly
     135             :  *   at the origin.
     136             :  * - \f$0 < r_B < r_A\f$
     137             :  * - \f$R \ge 3(|c_A^0|-|c_B^0|)\f$; otherwise the blocks will be too compressed
     138             :  *   near the outer boundary.
     139             :  *
     140             :  */
