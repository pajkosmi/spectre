     296             : /*!
     297             :  * \brief %Matrix used to perform an indefinite integral of a function over the
     298             :  * logical grid. The left boundary condition is such that the integral is 0 at
     299             :  * \f$\xi=-1\f$
     300             :  *
     301             :  * Currently only Legendre and Chebyshev polynomials are implemented, but we
     302             :  * provide a derivation for how to compute the indefinite integration matrix for
     303             :  * general Jacobi polynomials.
     304             :  *
     305             :  * #### Legendre Polynomials
     306             :  * The Legendre polynomials have the identity:
     307             :  *
     308             :  * \f{align*}{
     309             :  * P_n(x) = \frac{1}{2n+1}\frac{d}{dx}\left(P_{n+1}(x) - P_{n-1}(x)\right)
     310             :  * \f}
     311             :  *
     312             :  * The goal is to evaluate the integral of a function \f$u\f$ expanded in terms
     313             :  * of Legendre polynomials as
     314             :  *
     315             :  * \f{align*}{
     316             :  * u(x) = \sum_{i=0}^{N} c_i P_i(x)
     317             :  * \f}
     318             :  *
     319             :  * We similarly expand the indefinite integral of \f$u\f$ as
     320             :  *
     321             :  * \f{align*}{
     322             :  * \left.\int u(y) dy\right\rvert_{x}=&\sum_{i=0}^N \tilde{c}_i P_i(x) \\
     323             :  *   =&\left.\int\sum_{i=1}^{N}\frac{c_i}{2i+1}
     324             :  *      \left(P_{i+1}(y)-P_{i-1}(y)\right)dy\right\rvert_{x}
     325             :  *      + \tilde{c}_0 P_0(x) \\
     326             :  *   =&\sum_{i=1}^{N}\left(\frac{c_{i-1}}{2i-1} - \frac{c_{i+1}}{2i+3}\right)
     327             :  *     P_i(x) + \tilde{c}_0 P_0(x)
     328             :  * \f}
     329             :  *
     330             :  * Thus we get that for \f$i>0\f$
     331             :  *
     332             :  * \f{align*}{
     333             :  * \tilde{c}_i=\frac{c_{i-1}}{2i-1}-\frac{c_{i+1}}{2i+3}
     334             :  * \f}
     335             :  *
     336             :  * and \f$\tilde{c}_0\f$ is a constant of integration, which we choose such that
     337             :  * the integral is 0 at the left boundary of the domain (\f$x=-1\f$). The
     338             :  * condition for this is:
     339             :  *
     340             :  * \f{align*}{
     341             :  *   \tilde{c}_0=\sum_{i=1}^{N}(-1)^{i+1}\tilde{c}_i
     342             :  * \f}
     343             :  *
     344             :  * The matrix returned by this function is the product of the tridiagonal matrix
     345             :  * for the \f$\tilde{c}_i\f$ and the matrix for the boundary condition.
     346             :  *
     347             :  * #### Chebyshev Polynomials
     348             :  *
     349             :  * A similar derivation leads to the relations:
     350             :  *
     351             :  * \f{align*}{
     352             :  *  \tilde{c}_i=&\frac{c_{i-1}-c_{i+1}}{2i},&\mathrm{if}\;i>1 \\

     358             :  * \f{align*}{
     359             :  * \tilde{c}_0=\sum_{i=1}^N(-1)^{i+1}\tilde{c}_i
     360             :  * \f}
     361             :  *
     362             :  * These are then used to define the indefinite integration matrix.
     363             :  *
     364             :  * #### Jacobi Polynomials

     369             :  *  (1-x)^\alpha(1+x)^\beta P^{(\alpha,\beta)}_n(x)=\frac{(-1)^n}{2^n n!}

     378             :  * b^{(\alpha,\beta)}_{n,n}P^{(\alpha,\beta)}_n(x) +

     380             :  * \right)

     383             :  * where
