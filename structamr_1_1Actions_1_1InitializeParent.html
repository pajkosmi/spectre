Detailed Description

Initializes the data of a newly created parent element from the data of its children elements.

Warning
At the moment, this action only initializes the Element, Mesh, and amr::Flags of the parent element. It does not initialize any data related to evolution or elliptic solves.

Details

This action is meant to be invoked by amr::Actions::CollectDataFromChildren
