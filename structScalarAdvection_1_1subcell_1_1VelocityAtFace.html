Detailed Description

template<size_t Dim>
struct ScalarAdvection::subcell::VelocityAtFace< Dim >

Allocate and set the velocity field needed for evolving ScalarAdvection system when using a DG-subcell hybrid scheme.

Uses:

  • DataBox:
    • Tags::Time
    • evolution::dg::subcell::Tags::Mesh<Dim>
    • domain::Tags::ElementMap<Dim, Frame::Grid>
    • Tags::CoordinateMap<Dim, Frame::Grid, Frame::Inertial>
    • domain::Tags::FunctionsOfTime
    • evolution::dg::subcell::Tags::Coordinates<Dim, Frame::ElementLogical>

DataBox changes:

  • Adds:
    • evolution::dg::subcell::Tags::Inactive<velocity_field>
    • evolution::dg::subcell::Tags::OnSubcellFaces<velocity_field, Dim> where velocity_field is ScalarAdvection::Tags::VelocityField<Dim>.
  • Removes: nothing
  • Modifies: nothing
Note
This mutator is meant to be used with Initialization::Actions::AddSimpleTags.
