Detailed Description

template<typename Metavariables>
class Parallel::MutableGlobalCache< Metavariables >

A Charm++ chare that caches mutable data once per Charm++ core.

MutableGlobalCache is not intended to be visible to the end user; its interface is via the GlobalCache member functions mutable_global_cache_proxy, mutable_cache_item_is_ready, mutate, and get. Accordingly, most documentation of MutableGlobalCache is provided in the relevant GlobalCache member functions.

Note
Very seldomly will a user need a proxy to the MutableGlobalCache. If you think that you need it, please consult a core developer to see if there is a better way to achieve what you are trying to do.
